## ğŸš€ Gameplay Flow

1. Click the rocket to launch  
2. Watch rocket ascend  
3. Result calculated on ascent completion  
4. Loss triggers explosion animation  
5. Win triggers popup  
6. History panel updates automatically  
7. Rocket resets for next round  
8. Unlimited rounds, play freely
